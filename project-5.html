The Calculator

Project 5

The title of this project says it all. The task was to create a calculator. Flexbox was used for button placement. Sizing and margins for both the buttons and the screen was done using em units, allowing the size of the calculator to be scalable by changing the font size. Buttons were given box-shadows and transitions to simulate button presses.
